## Abrir sem servidor local

Este projeto usa Vite + React. Para abrir no navegador sem `npm run dev`:

1. Gere os arquivos finais:

```bash
npm run build
```

2. Abra o arquivo `dist/index.html` com duplo clique.

O build foi configurado com caminhos relativos, entao funciona em `file://`.
